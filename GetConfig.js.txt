
How to Make Ai token work for test.html +

In project main, create a new file named config.js

Inside that config.js file, paste this code (edit the API key):



// This is the config.js file they need to create
const OPENROUTER_API_KEY = "PASTE-THEIR-OWN-API-KEY-HERE";

export { OPENROUTER_API_KEY };




As soon as you do that, the PDF feature will work.
